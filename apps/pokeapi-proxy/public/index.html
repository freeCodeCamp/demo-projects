PokéAPI Proxy

Usage

Use the endpoint https://pokeapi-proxy.freecodecamp.rocks/api/pokemon/{name-or-id} to get data for a Pokémon, where {name-or-id} is the Pokémon's name or its id number in the official Pokédex.

Note that the name should be lowercase, have special characters removed, and dash separated.

Example Requests

Click any of the example requests below to see its response.

  • https://pokeapi-proxy.freecodecamp.rocks/api/pokemon/pikachu
  • https://pokeapi-proxy.freecodecamp.rocks/api/pokemon/25
