Fruit Search API

Usage

Use the endpoint https://fruit-search.freecodecamp.rocks/api/fruits?q={query} to filter through fruits, where query is your query parameter.

Example Requests

  • https://fruit-search.freecodecamp.rocks/api/fruits?q=a returns a list of fruits containing the letter a.
  • https://fruit-search.freecodecamp.rocks/api/?q=berry returns a list of fruits containing the word berry.
