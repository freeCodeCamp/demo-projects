freeCodeCamp Weather API Passthrough

USAGE:
Use the endpoint: https://fcc-weather-api.freecodecamp.repl.co/. Use this endpoint to get the weather at a location. To prevent abuses this server accepts GET requests only, and serves only the route /api/current?lon=:longitude&lat=:latitude. Images links are included in the JSON under weather[0].icon. This is enough to complete the challenge.
Example:
Request: https://fcc-weather-api.freecodecamp.repl.co/api/current?lat=35&lon=139
Response: { "coord": { "lon": 139, "lat": 35 }, "weather": [ { "id": 801, "main": "Clouds", "description": "few clouds", "icon": "https://fcc-weather-api.freecodecamp.repl.co/images/02n.png" } ], "base": "stations", "main": { "temp": 25, "feels_like": 28.82, "temp_min": 25, "temp_max": 25, "pressure": 1010, "humidity": 84 }, "visibility": 10000, "wind": { "speed": 1.34, "deg": 177, "gust": 2.68 }, "clouds": { "all": 16 }, "dt": 1597922946, "sys": { "type": 3, "id": 2019346, "country": "JP", "sunrise": 1597867673, "sunset": 1597915636 }, "timezone": 32400, "id": 1851632, "name": "Shuzenji", "cod": 200 }
